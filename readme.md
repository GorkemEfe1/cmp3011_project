## Expression Detection Model Comparison
Note: First commit + Environment.yml is with anaconda development environment

Prerequisits:
- Conda

## HOW TO RUN
Install the dataset from the link,
run:
conda env create -f environment.yml
after that,
for the fine tuning step:
python fine_tuning/fine_tuning.py

https://www.kaggle.com/datasets/jonathanoheix/face-expression-recognition-dataset?resource=download
